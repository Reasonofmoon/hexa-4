## 🔵 Professional: AI 진단 프롬프트 10개

### 프롬프트 1: 개인화 학습 경로 설계
```
당신은 교육 AI 전문가입니다.
[K-12 온라인 교육 플랫폼]의 학습 데이터를 분석하여 개인화 학습 경로를 설계해주세요.
- 현재 학습 완료율: [65%]
- 학습 성취도 편차: [±25%]
- 목표: 완료율 85% 이상, 성취도 편차 ±10% 이내
학습 스타일 분석과 적응형 학습 콘텐츠 추천 시스템을 구축해주세요.
```

### 프롬프트 2: 지능형 평가 시스템
```
당신은 교육 평가 AI 전문가입니다.
[대학 교육 플랫폼]의 평가 시스템을 분석하여 지능형 평가 모델을 제시해주세요.
- 현재 평가 신뢰도: [75%]
- 채점 시간: [답당지당 3분]
- 목표: 신뢰도 90% 이상, 채점 시간 30초 이내
자동 채점 시스템과 다차원 평가 분석 모델을 구축해주세요.
```

### 프롬프트 3: 학습 부진자 조기 발견
```
당신은 교육 데이터 분석 AI 전문가입니다.
[어학원]의 학습 데이터를 분석하여 학습 부진자 조기 발견 시스템을 설계해주세요.
- 현재 중도탈락률: [18%]
- 조기 발견율: [40%]
- 목표: 중도탈락률 8% 이하, 조기 발견율 80% 이상
학습 패턴 분석과 위험 신호 탐지 알고리즘을 개발해주세요.
```

### 프롬프트 4: 지능형 콘텐츠 추천
```
당신은 에드테크 콘텐츠 AI 전문가입니다.
[직업 교육 플랫폼]의 콘텐츠를 분석하여 지능형 추천 시스템을 구축해주세요.
- 현재 콘텐츠 활용률: [45%]
- 학습자 만족도: [72점/100점]
- 목표: 활용률 75% 이상, 만족도 85점 이상
학습자 프로필 기반 콘텐츠 매칭 알고리즘을 설계해주세요.
```

### 프롬프트 5: 자동 과제 생성 및 채점
```
당신은 교육 자동화 AI 전문가입니다.
[프로그래밍 교육 플랫폼]의 과제 시스템을 분석하여 자동화 방안을 제시해주세요.
- 현재 과제 생성 시간: [주당 4시간]
- 피드백 제공 시간: [과제당 15분]
- 목표: 생성 시간 30분 이내, 피드백 1분 이내
난이도 조절 자동 과제 생성과 실시간 피드백 시스템을 구축해주세요.
```

### 프롬프트 6: 학습 효과 측정 및 분석
```
당신은 교육 평가 AI 전문가입니다.
[기업 교육 플랫폼]의 학습 효과를 측정할 수 있는 AI 시스템을 설계해주세요.
- 현재 효과 측정 정확도: [60%]
- 지식 보존율: [3개월 후 45%]
- 목표: 측정 정확도 85% 이상, 지식 보존율 70% 이상
다차원 학습 분석과 장기 기억 모델링 시스템을 구축해주세요.
```

### 프롬프트 7: 가상 튜터 시스템
```
당신은 교육 AI 전문가입니다.
[개인 과외 플랫폼]을 위한 가상 튜터 시스템을 설계해주세요.
- 현재 1:1 튜터 비용: [시간당 5만원]
- 학습자 만족도: [68점/100점]
- 목표: 비용 80% 절감, 만족도 85점 이상
자연어 대화와 개인화 교수법을 구현한 AI 튜터를 개발해주세요.
```

### 프롬프트 8: 학습 동기 부여 시스템
```
당신은 교육 동기부여 AI 전문가입니다.
[청소년 교육 플랫폼]의 학습 동기 부여 전략을 수립해주세요.
- 현재 학습 지속율: [주 3.2회]
- 학습 참여도: [55%]
- 목표: 지속율 주 5회 이상, 참여도 80% 이상
게이미피케이션과 보상 시스템을 결합한 동기부여 AI를 설계해주세요.
```

### 프롬프트 9: 지식 맵핑 및 연관성 분석
```
당신은 교육 지식 구조 AI 전문가입니다.
[인문학 교육 플랫폼]의 지식 구조를 분석하여 연관성 모델을 구축해주세요.
- 현재 지식 연결 정확도: [65%]
- 학습자 이해도: [58%]
- 목표: 연결 정확도 90% 이상, 이해도 80% 이상
개념 그래프와 지식맵 시각화 시스템을 개발해주세요.
```

### 프롬프트 10: 교육 품질 모니터링
```
당신은 교육 품질 AI 전문가입니다.
[온라인 강의 플랫폼]의 교육 품질을 모니터링할 AI 시스템을 설계해주세요.
- 현재 품질 평가 주기: [월 1회]
- 품질 개선 사이클: [평균 3개월]
- 목표: 실시간 모니터링, 개선 사이클 2주 이내
강의 품질 자동 평가와 실시간 개선 제언 시스템을 구축해주세요.
```
